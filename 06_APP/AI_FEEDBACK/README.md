# AI Feedback

Usa esta carpeta para pegar o guardar respuestas de otras IAs sobre el motor.

Formato recomendado:
- un archivo `.txt` por IA o por sesion;
- nombres simples y claros.

Ejemplos:
- `claude_motor_auditoria_01.txt`
- `gemini_mejoras_motor_01.txt`
- `grok_ideas_modelo_01.txt`
- `perplexity_contexto_modelo_01.txt`

Consejo:
- si una respuesta trae varias propuestas, no la edites demasiado;
- mejor pegala casi completa para poder compararla despues.

Objetivo:
- leer varias auditorias del motor;
- comparar ideas;
- detectar patrones repetidos;
- convertir lo mejor en mejoras reales del sistema.
