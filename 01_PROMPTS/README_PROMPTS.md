# Organizacion de Prompts

Esta carpeta separa los prompts por uso real:

- `manuales/`
  - Prompts largos y explicativos.
  - Pensados para trabajar uno a uno en Perplexity, ChatGPT, Gemini, Claude, etc.
  - Buenos para revisar calculos, contexto y razonamiento completo.

- `automatizacion/`
  - Prompts compactos y estrictos.
  - Pensados para APIs y ejecucion multi-IA dentro de la app.
  - Priorizan velocidad, consistencia JSON y menor consumo de tokens.

Notas:

- `scout_prompt.txt` y `juez_prompt.txt` pueden seguir en la raiz por ahora.
- Los archivos originales no se eliminan. Estas carpetas agregan una estructura mas clara.
